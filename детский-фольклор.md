## Колыбельные Песни

**№ 1. Ой, лю-лю, моё дитятко**

```
Ой, лю-лю, моё дитятко
Спи-тко, усни, дитя материно,
Все ласточки спят,
И касатки спят,
Куницы спят,
И лисицы спят,
Нашему Ванюше
Спать велят.
Для чего, зачем
Ванюше не спать?
Ласточки спят
Все по гнёздышкам,
Касаточки спят
Все по кусточкам,
Лисицы спят
Все по подкусточкам,
Куницы спят
Все по норочкам,
Соколы спят
Все по гнёздышкам,
Соболи спят,
Где им вздумалось,
Маленькие детки
В колыбелях спят.
Спи-тко, Ванюша,
Спи-тко, дитятко родное!
```
